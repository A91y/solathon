## Classes
#### AccountMeta
This is a dataclass reperenting meta-data of an account

<Code>
```python
@dataclass
class AccountMeta:
    public_key: PublicKey | str
    is_signer: bool
    is_writable: bool
```
</Code>

#### Instruction
This is a named tuple reperenting a instruction object. Here keys are AccountMeta dataclass, program_id is the PublicKey of the program and data is our compiled instruction in bytes.

<Code>
```python
class Instruction(NamedTuple):
    keys: list[AccountMeta]
    program_id: PublicKey
    data: bytes = bytes(0)
```
</Code>
