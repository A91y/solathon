### Feature rich
Solathon covers most of the Solana JSON RPC. Other than that, it provides utility functions which come in handy often.

### High performance
Being light weight and optimized for the best performance; Solathon comes with batteries included without being too heavy.

### Easy to use
Everything has been simplified down for it to be easy to use, using object-oriented approach, Solathon makes everything easy to understand and work with.

### Async support
Solathon supports both synchronous and asynchronous clients, use the one more suitable for your application.
